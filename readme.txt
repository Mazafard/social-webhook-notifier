Send a JSON webhook when a post is published, so you can automate publishing to social media via n8n, Zapier, etc.

== Description ==

A simple plugin to send a webhook when a post is published, including title, link, excerpt, and author. Perfect for integration with n8n or Zapier.

== Installation ==

1. Upload the plugin zip file to the `wp-content/plugins/` directory.
2. Activate the plugin through the ‘Plugins’ menu.
3. Set your webhook URL in `Settings → Social Webhook Notifier`.
